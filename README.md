  # CSharp Expense Tracker

A simple console-based Expense Tracker application built using C# and .NET Framework 4.7.2.

## Features
- Add expense
- View all expenses
- Search expenses by category
- Calculate total expense
- Delete expense
- Save data using text file storage

## Technologies Used
- C#
- .NET Framework 4.7.2
- LINQ
- File Handling
- OOP
- Collections

## Project Purpose
This project was created to practice core C# programming concepts and build a simple real-world console application.
